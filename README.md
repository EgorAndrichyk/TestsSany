Дальше будет список действий для запуска моего мега-приложения:
1. Для начала прошу клонировать репозиторий.
2. Затем нужно будет установить виртуальное окружение с помощью команды `python3 -m venv .venv`.
3. Активировать и установить зависимости(если удобно то через **uv**).
4. Сохранить тестовые данные в папке csv.
5. Запустить main.py
6. Ввести лимит средств, который запланирован на повышение оплат.
