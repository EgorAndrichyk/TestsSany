### TestsSany
### Добрый день, Александр 
Сегодня я презентую свою версию приложения для расчета повышения оплаты в компании на тысячи человек, только достойный получил прибавку(я тут решаю кста).
Дальше будет список действий для запуска моего мега-приложения через файл:
1. Для начала прошу клонировать репозиторий https://github.com/EgorAndrichyk/TestsSany.git.
2. Затем нужно будет установить виртуальное окружение с помощью команды
   ```bash
   uv venv .venv
   ```
4. Активировать и установить зависимости
   ```bash
    source venv/bin/activate
   ```
   
   ```bash
    uv sync
   ```
6. Сохранить тестовые данные в папке csv(да нужно создать к сожалению, такая душка).
7. Запустить main.py
   
   ```bash
   uv run main.py
   ```
   
8. Ввести лимит средств, который запланирован на повышение оплат.

Теперь будем запускать через фреймворк Django(с 1 по 5 пункт первого задания, повторяем):


6. Применяем миграции

    ```bash
    python manage.py migrate
    ```
    
7. Запускаем проект

    ```bash
    python manage.py runserver
    ```
8. Переходим по ссылке

    ```bash
    http://127.0.0.1:8000/
    ```
