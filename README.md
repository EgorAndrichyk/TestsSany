### TestsSany
### Добрый день, Александр 
Сегодня я презентую свою версию приложения для расчета повышения оплаты в компании на тысячи человек, только достойный получил прибавку(я тут решаю кста).
Дальше будет список действий для запуска моего мега-приложения:
1. Для начала прошу клонировать репозиторий https://github.com/EgorAndrichyk/TestsSany.git.
2. Затем нужно будет установить виртуальное окружение с помощью команды
   > `python3 -m venv .venv`.
4. Активировать(`source venv/bin/activate`) и установить зависимости
   > `python -m pip install --upgrade pip`
   
   > `pip install -r requirements.txt`
5. Сохранить тестовые данные в папке csv(да нужно создать к сожалению, такая душка).
6. Запустить main.py
7. Ввести лимит средств, который запланирован на повышение оплат.
